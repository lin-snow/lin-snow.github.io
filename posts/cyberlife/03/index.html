这段时间又折腾了些新玩意，又有素材可以水啦。欢迎来到L1nSn0wの二进制生活！😁

📷博客配图心得

评论区经常有小伙伴问我博客的配图是怎么做的，所以今天就分享一些我的配图小技巧。对于我来说，绝大部分图片都来自手机拍摄与屏幕截图，所以当我需要拍一些图片时，我一般都是直接打开手机的第三方软件进行拍摄，原因是使用第三方软件拍摄对比iOS的原相机更方便，大部分软件都能提供丰富的滤镜，不管是拍摄物品还是景色，基本上都能够找到心仪的滤镜（或者“胶卷”），对于我这种又懒又不会调色的纯小白算是非常实用了，平时写博客的时间非常少，自然不能在图片上花费过多的时间，效率至上！

当遇到需要截屏的场景时，这种情况也很容易处理，最简单当然是直接使用顺手的截图工具进行截图。不过我一般是截图完之后会给图片套个壳，个人认为套上壳之后看起来会更精致些。这里的套壳当然也是借助网站来解决，我个人用得最多最推荐是postspark ,这个网站功能强大丰富，而且也是免费的。

最后列一些我常用的手机拍摄软件：

  • Fimo (超多胶卷)
  • 极影相机 （免费，功能丰富）
  • Leica Lux (徕卡官方App,仅用免费的徕卡滤镜和徕卡水印)

其中Fimo和极影相机这两个软件我在之前的文章也有提到过，感兴趣的话也可以去看看☺️

🙃论坛分享

平时各种折腾还得靠论坛大佬的教程和资源保驾护航，所以打算分享几个个人常逛的论坛：

  • 远景论坛: 相信玩过黑苹果的人都知道这个论坛，这个应该是国内最大的黑苹果论坛了，上面有各种大佬分享EFI和镜像，而且黑苹果过程中遇到的问题基本上都能在论坛上找到解决方法，可以说玩黑苹果必备了。不过话又说回来，黑苹果在Apple转向m系列芯片之后基本上也走向死亡了，不过我还是想分享一下这个属于黑苹果时代的宝藏论坛！（黑苹果已死😭😭😭）

  • 恩山论坛：如果你喜欢折腾各种路由器和Openwrt，那么恩山论坛必不能错过，这个论坛上经常会有大佬分享各种编译好的wrt固件，并且论坛上的帖子非常详细，像我这样完全没接触过这些领域的纯小白靠着论坛上的帖子最后也成功给AX3000T刷上了ImmortalWRT😇

💻系统大瘦身

自从上次给C盘扩容后，我安装软件的策略也发生了改变，虽然说C盘变大了，但是对于软件的安装我依然保持谨慎，吸取上次的教训，我没有在系统重装完之后安装大量的软件，我只安装了能够最大满足我需求的软件，并且功能重叠较大的软件则保留下使用最顺手的一款。对于Adobe家的软件，我没有安装Lr和Pr，只保留了最常用的PS，一方面是我基本上不剪视频，也不调色（虽然想学），另一方面则是怕Adobe家的软件在我系统里到处拉屎。

在仔细分析我的开发需求后，我只保留了VSCode和Jetbrain 家的IntelliJ IDEA Ultimate，这两个软件一个是编辑器，一个是适合Java的IDE，不过已经能够完全满足我的前后端开发需求（安装完插件后）。不过这里还是得吐槽一下VSCode的Java开发体验，不夸张的说就算VSCode装好各种插件体验依然被IDEA甩好几条街，本来以为可以All in VScode的，现在看来是我想多了🤣

🛜AX3000T & Immortal

没错，我又折腾上了我的路由器，相比于我的赛博生活01那篇博客里不同的是，这一次我给AX3000T刷上了自己编译定制的ImmortalWRT。

固件仅支持小米AX3000T v1 !基于hanwckf大佬的immortalwrt-mt798x进行编译，同样使用h大提供的的mt798x uboot进行刷入引导。

固件特点：

  1. 内核版本：5.4.284。 Openwrt版本： ImmortalWrt 21.02-SNAPSHOT
  2. 小闪存友好（整个固件仅19.2MB）
  3. 只保留OC用于支持爱国上网
  4. 移除原固件不必要的bootstrap mod主题
  5. 支持Upnp和友好的ipv6-helper
  6. …

我已经将编译好的固件分享到了恩山论坛，原贴地址：传送门

☕JetBrain续杯啦

前几天IDEA突然给我弹了个非常陌生的通知，没错，我的IDEA许可证将在二十几天后过期，看到这条消息的我愣了一下，毕竟springboot正写的火热，你给我弹这个消息。至从上次学生认证完之后我就再也没有去管过许可证的到期时间了，一直以为是可以直接用到毕业，没想到是每隔一年要续一次，还好我的学生证还在我身边，学信网的在线验证码也很容易获取，于是我立马提交了续订资料，向之前一样，这次依然走的人工通道，效率还行，大概花了三四天左右吧，比上次快了很多。这不，今天就收到了续订成功通知，爽歪歪！😁（IDEA YYDS!）

🦖Tika能用吗

为了能快速学习和巩固新的知识，我做了个功能非常简陋的Todo Web 应用，技术栈大概如下：

  • 前端： Vue、Nuxt.js、Nuxt UI、Typescript、Tailwind CSS …

在线体验地址： https://tika.soopy.cn/

⌚小米手环9用户+1

今年春节淘宝的红包真的很大，总共是薅到了52.82元的通用红包，加上手表国补15%（补贴36.18）和小米店铺优惠券5元，最终以205元拿下心心念念已久的小米手环9 NFC版 银色！顺丰是真的快，晚上下单完隔天下午就到货。不过因为是国补，所以到货的时候还需要派送员现场激活拍照才行，手机还好，手表必须要插电才能开机，所以还是有点小麻烦的。

目前已经连续带了有好几天了，总体来说体验非常好，60hz的刷新率使用起来丝般顺滑，续航真的顶，在开启大部分功能的情况下可以撑9天，而且佩戴起来也非常舒适，比我之前的华为荣耀手环5NFC版好太多了，各种功能和表盘也很丰富！

至于心率血氧什么的我也不知道测的准不准，不过无所谓，反正我也不指望一个小手环能有多精确，手环自带的功能和装饰性已经能够满足我的需求了。
